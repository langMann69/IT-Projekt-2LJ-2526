🎯 Projektziel

Ziel dieses Softwareprojekts ist die Entwicklung einer konsolenbasierten Wissensdatenbank für Hotline-Mitarbeiter.

Die Anwendung soll es ermöglichen, häufig gestellte Fragen (FAQs) mit zugehörigen Lösungen zu erfassen, zu speichern, zu durchsuchen, zu bearbeiten und zu löschen.

Dadurch soll die Bearbeitung von Kundenanfragen effizienter und strukturierter erfolgen.

📋 Product Backlog (User Stories)
🔹 Priorität 1 (hoch)

US1: Fragen mit Lösung erfassen und in Datei speichern

US2: Fragen über Suchbegriff finden und anzeigen

🔹 Priorität 2 (mittel)

US3: Bestehende Fragen bearbeiten

US4: Bestehende Fragen löschen

🔹 Priorität 3 (gering)

US5: Komfortable Eingabemaske zur fehlerfreien Datenerfassung

🛠 Technische Umsetzung

Programmiersprache: Python

Benutzerinteraktion: Konsole (CLI)

Datenspeicherung: Datei-basiert (z. B. JSON oder CSV)

Versionsverwaltung: Git / GitHub

Vorgehensmodell: Agile Softwareentwicklung mit 3 Sprints
