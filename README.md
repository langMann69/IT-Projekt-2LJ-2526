# IT-Projekt-2LJ-2526
Ziel dieses Softwareprojekts ist die Entwicklung einer konsolenbasierten Wissensdatenbank für Hotline-Mitarbeiter.  Die Anwendung soll es ermöglichen, häufig gestellte Fragen (FAQs) mit zugehörigen Lösungen zu erfassen, zu speichern, zu durchsuchen, zu bearbeiten und zu löschen.
